## Setup Instructions

### 1. Clone the repository

```bash id="cl1"
git clone https://github.com/edmarcloydj/my-portfolio.git
cd my-portfolio
```

### 2. Install dependencies

```bash id="cl2"
npm install
```

### 3. Run the project

```bash id="cl3"
npm run dev
```

### 4. Open in browser

http://localhost:3000
